开发环境部署

1. 配置 jdk

  1. 输入命令安装jdk

yum -y list java*
yum -y install java-1.8.0-openjdk.x86_64
等待安装jdk
安装完成后，输入java -version若显示jdk版本，则说明安装成功。

  1. 安装javac

yum -y install java-1.8.0-openjdk-devel
等待安装javac
安装完成后，输入javac -version若显示版本信息，则说明安装成功。

2. 配置 Go

  1. 前往 Go官网 下载Linux版本的压缩包并传入服务器
  2. 使用 tar -zxvf xxx.tar.gz 命令解压文件
  3. 使用 mv go /usr/local/ 将go文件夹放到/usr/local文件夹下
  4. 配置PATH：

vim /etc/profile
在文件结尾处加上以下代码：
export PATH=$PATH:/usr/local/go/bin
刷新profile文件：
source /etc/profile

  1. 输入 go version 若显示版本，则安装成功。

3. 配置 g++

  1. 输入 g++ -v 查看g++是否安装
  2. 若未安装，输入 yum install gcc-c++
  3. 查看g++是否安装成功

4. 配置 Tomcat

  1. 前往Tomcat官网，复制Tomcat压缩包下载链接（后缀为.tar.gz）
  2. 输入命令 wget https://dlcdn.apache.org/tomcat/tomcat-9/v9.0.53/bin/apache-tomcat-9.0.53.tar.gz （地址为你所复制的下载链接）
  3. 下载完成后输入命令 tar -zxvf apache-tomcat-9.0.53.tar.gz （压缩包名为你下载的tomcat包名）
  4. 将tomcat移动到/usr/local目录下 mv apache-tomcat-9.0.53.tar.gz /usr/local
  5. 进入apache-tomcat-9.0.53/bin目录，输入命令 ./startup.sh 启动tomcat服务。
  6. 浏览器地址栏输入主机地址:8080查看是否能访问Tomcat欢迎页（注意查看服务器安全组是否开启了8080端口）

    若显示如上页面，则启动成功。
  7. 修改tomcat/bin的catalina.sh文件，添加如下内容 CATALINA_PID="$CATALINA_BASE/tomcat.pid"
  8. 在/usr/lib/systemd/system下面建立tomcat.service文件，文件内容为：
  1. 重新载入配置
    systemctl daemon-reload
  2. 添加自启动
    systemctl enable tomcat.service
    （可以使用disable关闭）
  3. 查看是否自启动
    systemctl is-enabled tomcat.service

配置 Vim

  1. 显示行号
    1. 局部显示行号：
      在vim界面输入:set nu

      输入此命令后即可显示行号：
    2. 全局显示行号：
      进入全局配置vim /etc/vimrc
      在文件末尾加上 set nu 即可。
  2. 导入配置好的vimrc文件
    我用阿里云盘分享了「vimrc」，你可以不限速下载🚀

链接：https://www.aliyundrive.com/s/RcLMyk8EaeW
下载好之后上传到/etc 文件夹内，覆盖原有的vimrc文件即可。

vim配置功能如下：

  • 按F5可以直接编译并执行C、C++、java代码以及执行shell脚本，按“F8”可进行C、C++代码的调试
  • 自动插入文件头 ，新建C、C++源文件时自动插入表头：包括文件名、作者、联系方式、建立时间等，读者可根据需求自行更改
  • 映射“Ctrl + A”为全选并复制快捷键，方便复制代码
  • 按“F2”可以直接消除代码中的空行
  • “F3”可列出当前目录文件，打开树状文件目录
  • 支持鼠标选择、方向键移动
  • 代码高亮，自动缩进，显示行号，显示状态行
  • 按“Ctrl + P”可自动补全
  • []、{}、()、””、’ ‘等都自动补全
  • 其他功能可以研究配置文件

Code-Server

搭建Code-Server

  1. 前往Code-Server官网release仓库下载Linux-amd64版本。
    • Windows系统下载压缩包，然后使用FTP发送到服务器上
    • 服务器使用 wget www.xxx.com 直接下载压缩包
  2. 使用 tar xf code-server.tar.gz 解压缩
  3. 使用 vim ~/.config/code-server/config.yaml 修改配置文件


0.0.0.0表示任何人都能访问，后面xxxx表示端口号（需要去控制台安全组开启端口）
password字段填写登录VsCode的密码

  1. 在code-server目录下输入指令 ./code-server 开启code-server
  2. 在网址输入主机:端口号即可访问
