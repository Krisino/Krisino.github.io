开发环境部署指南

一、安装 Oracle JDK

  1. 获取 JDK 安装包

  • 前往 Oracle JDK 官网 获取 JDK 下载链接 url，运行命令 wget url

  • 手动在Oracle JDK官网下载，并上传到服务器

  1. 解压 JDK

运行命令 tar -zxvf jdk-8u300-linux-x64.tar.gz 进行解压（注意文件名替换为自己下载的文件名）

  1. 移动 JDK 文件夹

运行命令 cd /usr/local mkdir java mv jdk-8u300-linux-x64 /usr/local/java （第三条命令需先回到 JDK 解压目录）

运行命令 source /etc/profile

  1. 查看 JDK 是否安装成功

输入命令 java -version javac -version

二、安装 Go

  1. 获取 Go 安装包
  • 前往 Go官网 获取下载链接 url，运行命令 wget url
  • 手动在 Go 官网下载，并上传到服务器
  1. 解压 Go

运行命令 tar -zxvf xxx.tar.gz 解压文件

  1. 移动 Go 文件夹

使用 mv go /usr/local/ 将 go 文件夹放到 /usr/local 文件夹下

  1. 配置PATH：

运行命令 vim /etc/profile 在文件结尾处加上以下代码：export PATH=$PATH:/usr/local/go/bin

  1. 更新 profile 配置文件

运行命令 source /etc/profile

  1. 查看 Go 是否安装成功

运行命令 go version

三、安装 Tomcat

  1. 获取 Tomcat 压缩包
  • 前往 Tomcat官网 获取下载链接 url，运行命令 wget url
  • 手动在 Tomcat 官网下载，并上传到服务器
  1. 解压压缩包

下载完成后输入命令 tar -zxvf apache-tomcat-9.0.53.tar.gz

  1. 移动 Tomcat 文件夹

运行命令 mv apache-tomcat-9.0.53.tar.gz /usr/local 将 tomcat 移动到/usr/local目录下

  1. 进入 usr/local/apache-tomcat-9.0.53/bin 目录，输入命令 ./startup.sh 启动tomcat服务。

  2. 运行 Tomcat

浏览器地址栏输入 主机地址:8080 查看是否能访问 Tomcat 欢迎页（如果是服务器注意查看安全组是否开启了 8080 端口）

  1. 设置 Tomcat 开机自启

修改 tomcat/bin/catalina.sh 文件，添加如下内容 CATALINA_PID="$CATALINA_BASE/tomcat.pid"
