插入排序

插入排序 Insertion Sort

Algorithm

数据序列分为已排序和未排序两部分，其中每次排序对未排序部分中的一个元素插入到已排序的合适位置中，直到不存在未排序的元素，则该数据序列就排序完成。

类似于扑克牌中，每抓取一张扑克牌将其放在手中牌的合适位置，当所有牌都在手中时，则手中的牌为有序。

Input: <a1,a2,a3,…,an>\lt a_1,a_2,a_3,\dots,a_n \gt<a1​,a2​,a3​,…,an​>

Output: <a1′,a2′,a3′,…,an′>\lt a'_1,a'_2,a'_3,\dots,a'_n \gt<a1′​,a2′​,a3′​,…,an′​> Meets a1′≤a2′≤a3′≤⋯≤an′a'_1\leq a'_2\le a'_3\leq\dots\leq a'_na1′​≤a2′​≤a3′​≤⋯≤an′​

Analysis

  • 插入排序是稳定排序算法

  • 时间复杂度 O(n2)O(n^2)O(n2)

最好情况 最坏情况
比较次数 n - 1
移动次数 0
时间复杂度 O(n)O(n)O(n) O(n2)O(n^2)O(n2)
  • 空间复杂度 O(1)O(1)O(1)
