冒泡排序

冒泡排序 Bubble Sort

Algorithm

冒泡排序一共要进行若干轮冒泡。

每一轮中从第一个元素开始，比较该元素和后一个元素的大小。若前面元素大于后面元素，则交换它们，该过程直到最后一个元素比较完为止。

在之后的每一轮都进行同样的操作。

在进行 n 轮后，每一个元素都排序完成。

算法优化：

如果在某一轮比较中，元素之间的相对位置没有发生变化（没有交换），那么说明整个数据序列已经有序，无需继续下一轮的冒泡。

可以在每一轮设置一个标志位，若发生了交换，则改变标志位的值。若一轮冒泡下来标志位没有发生变化，则可以跳出循环。

Analysis

  • 冒泡排序不是稳定排序算法

  • 时间复杂度

最好情况 最坏情况
比较次数 n - 1
移动次数 0
时间复杂度 O(n)O(n)O(n) O(n2)O(n^2)O(n2)
  • 空间复杂度, 仅用一个临时变量来提供交换
