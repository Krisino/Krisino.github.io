Vue.js 2.x 使用指南

Vue是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

插值表达式

数据绑定最常用的方式是使用插值表达式 {{}}, 两个大括号中填写变量名

{{}} 大括号内部可以写 JavaScript 表达式，不能写语句和控制流代码

输出HTML

使用插值表达式只能原样输出 message 的文本内容，若 message 中是一个 html 标签，如何让其以 html 的类型来显示呢？应该使用 v-html 指令

修改 attribute

想要修改 html 标签的 attribute, 应该使用 v-bind 指令
